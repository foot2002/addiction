# 중독 응급처치 챗봇

독성물질 노출시 초기대응 정보를 제공하는 챗봇 기반 모바일 웹 애플리케이션입니다.

## 주요 기능

### 🏠 홈
- 응급상황 신고 버튼 (119, 1339)
- 챗봇 상담 시작
- 다빈도 노출물질 응급처치법 보기
- 병원 찾기
- 정보 검색

### 💬 챗봇 상담
- 단계별 상담 프로세스
- 노출물질 선택
- 증상 선택 및 평가
- 응급처치 정보 제공
- 응급상황 자동 판단

### 📋 응급처치 가이드
- 다빈도 중독물질별 상세 가이드
- 각 물질별 단계별 응급처치법
- 주의사항 및 경고
- 챗봇 상담 연계

### 🏥 병원 찾기
- 주변 응급실 검색
- 응급 연락처 (119, 1339)

## 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: 모바일 퍼스트 반응형 디자인
- **JavaScript (ES6+)**: 챗봇 로직 및 인터랙티브 기능
- **PWA 지원**: 오프라인 사용 가능 (manifest.json)

## 사용 방법

1. 웹 브라우저에서 `index.html` 파일을 엽니다
2. 또는 로컬 서버를 실행하여 사용합니다:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js (http-server)
   npx http-server -p 8000
   ```
3. 모바일 브라우저에서 접속하여 사용합니다

## 파일 구조

```
addiction-mobile/
├── index.html          # 메인 HTML 파일
├── styles.css          # 스타일시트
├── app.js              # JavaScript 애플리케이션 로직
├── data.js             # 중독물질별 응급처치 데이터
├── manifest.json       # PWA 매니페스트
└── README.md           # 프로젝트 문서
```

## 챗봇 사용 흐름

1. **상담 시작**: 홈 화면에서 "체크하고 상담하세요" 버튼 클릭
2. **노출물질 선택**: 챗봇이 제공하는 버튼에서 노출된 물질 선택
3. **증상 선택**: 현재 나타나는 증상 선택 (선택사항)
4. **응급처치 정보 제공**: 선택한 물질과 증상에 맞는 응급처치법 제공
5. **응급상황 판단**: 위험 증상 감지시 즉시 119 신고 안내

## 중요 안내

- 본 챗봇은 약물·농약·화학제품 등 독성물질 노출시 초기대응 정보를 제공합니다.
- 입력된 노출물질과 증상을 바탕으로 정보를 제공하고, 필요시 119 신고 및 응급의료기관 안내를 제공합니다.
- 정확한 물질명(또는 제품명)을 확인해 주세요.
- 호흡곤란·의식저하·경련·대량섭취·흡입시 즉시 119 신고 또는 응급실로 이동하십시오.
- 본 정보는 의료전문가의 진단·치료를 대체하지 않으며, 참고용으로만 제공됩니다.
- 최종판단과 책임은 사용자에게 있습니다.
